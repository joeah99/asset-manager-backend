# Introduction 
Backend Repo for the Asset Manager. Running .NET version 8. 

# How to run the Web API
1.	Clone repo to local machine.
2.	Open a new terminal and navigate into the API folder. --> CD Asset-Manager-Backend/API
3.	Type "Dotnet build" & hit enter, then type "Dotnet run" & hit enter. 
4.	Navigate to "https://localhost:5001/"
